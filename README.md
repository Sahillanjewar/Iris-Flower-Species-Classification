# Iris-Flower-Species-Classification
Machine Learning project that classifies Iris flower species using supervised learning algorithms in Python. Includes exploratory data analysis, visualization, and model evaluation.
# Iris Flower Species Classification

## Project Overview

This project uses Machine Learning techniques to classify Iris flowers into three species based on their physical characteristics.

The dataset contains measurements of sepal length, sepal width, petal length, and petal width for different Iris flowers. Using these features, the model predicts the species of the flower.

---

## Dataset

The dataset includes the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Variable:
- Species (Setosa, Versicolor, Virginica)
